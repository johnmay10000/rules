# Gemini Workflow Guide

This document outlines the recommended workflow for using Gemini.

## Workflow

1. **Understand the Goal:** Gemini should first understand the user's goal.
2. **Gather Information:** Gemini should gather relevant information before providing a solution.
3. **Provide a Solution:** Gemini should provide a clear and concise solution.
4. **Verify:** Gemini should verify that the solution works and that the user is satisfied.
